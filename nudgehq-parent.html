Getting Your API Token

2 minutes. Use your parent observer account.

1
Log into your observer account

Go to lakota.instructure.com with your parent credentials.

2
Open Account Settings

Click your profile picture or initials → Settings.

3
Find Approved Integrations

Scroll down → + New Access Token. Name it "Nudge HQ", leave expiry blank → Generate Token.

4
Copy immediately

Canvas shows the token only once. Copy it now and paste into the sidebar.

⚠️ Treat this like a password. It gives read access to Canvas data visible from your observer account.

Firebase Setup

This lets his check-offs sync instantly to your parent view. Free Google account required. Takes about 10 minutes.

1
Go to Firebase Console

Visit console.firebase.google.com and sign in with your Google account.

2
Create a new project

Click Add project → name it "NudgeHQ" → disable Google Analytics (not needed) → Create project.

3
Add a Realtime Database

In the left sidebar → Build → Realtime Database → Create Database. Choose United States location. Select Start in test mode (allows read/write for 30 days — you can lock it down later).

4
Copy your Database URL

After creation you'll see a URL like:

https://nudgehq-default-rtdb.firebaseio.com

Copy this into the Firebase Database URL field in the sidebar.

5
Get your API Key

In Firebase Console → click the ⚙️ gear icon → Project Settings. Under "Your apps", click the </> (web) icon to register a web app. Name it "NudgeHQ". Copy the apiKey value from the config snippet shown.

6
Paste both into the sidebar

Enter the Database URL and API Key in the sidebar fields, then click Connect Firebase. You should see the status turn green.

✅ Once connected, his check-offs will appear on your dashboard in real time — no refreshing needed.

Twilio SMS Setup

One-time setup, ~10 minutes. Free trial included.

1
Create a free account

twilio.com/try-twilio — no credit card needed.

2
Get a free phone number

Console → Phone Numbers → Manage → Buy a Number. Filter by SMS.

3
Verify his number (trial only)

Phone Numbers → Verified Caller IDs → add his number. One-time confirmation code.

4
Copy Account SID and Auth Token

Both on your Twilio Console homepage.

SID: ACxxxxxxxxxxxxxxxxxxxx
🔒 Credentials stay in your browser only — sent directly to Twilio when you tap a nudge button.

Make.com Auto-Reminder

Sends automatic morning texts from real Canvas API data. Free Make.com account, ~20 min setup.

1
Schedule Trigger

Make.com → New Scenario → Schedule → Every Day at your chosen time.

2
HTTP: Fetch missing submissions
URL: https://lakota.instructure.com/api/v1/users/STUDENT_ID/missing_submissions
Method: GET
Headers:
  Authorization: Bearer YOUR_OBSERVER_TOKEN
Query: filter[]=submittable
3
Iterator → loop each missing item

Flow Control → Iterator on the response array.

4
Firebase: check if he already marked it done

HTTP → GET https://YOUR-DB.firebaseio.com/nhq_checked/{{id}}.json?auth=YOUR_KEY. If result is true, skip with a Filter.

5
Twilio: Send SMS

Connect Twilio → Send an SMS. Use message template below.

Message template:
Hey! Just checking in 💙 It looks like "{{name}}" hasn't been submitted in Canvas yet. No stress — you've totally got this. We're here if you need help! ❤️
✅ Step 4 means if he's already checked it off on his device, Make.com will skip texting about it. His check-offs actually reduce your noise.
